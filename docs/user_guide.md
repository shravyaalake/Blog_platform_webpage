AI Blog Generator – User Guide
-----------------------------------------------------------------
1. Introduction
-----------------------------------------------------------------
This tool lets you generate a complete blog article using AI.
You only need to:
- Enter a topic
- Select tone
- Select desired length
- Click Generate Blog
The AI produces a clean, readable blog instantly.
-----------------------------------------------------------------
2. Steps to Use
Step 1: Enter a topic
Examples:
- “Impact of AI on education”
- “Role of class representative”
- “Benefits of electric vehicles”

Step 2: Choose writing tone
- Neutral
- Casual
- Formal
- Persuasive

Step 3: Choose blog length
- ~600 words
- ~1000 words
- ~1500 words

Step 4: Click Generate Blog
A loading indicator appears.

Step 5: Read the generated blog
The blog appears on the left side in a scrollable container.

Step 6: Copy the blog (optional)
Click Copy to copy entire blog text.
-----------------------------------------------------------------
3. Troubleshooting
Blog not generating?
- Check internet connection
- Ensure backend is running
- Ensure API key is valid

Copy button not working?
- Browser may block clipboard access
- Try clicking blog area first, then copy
-----------------------------------------------------------------
4. Limitations
- Requires internet access
- Gemini API quota may limit usage
